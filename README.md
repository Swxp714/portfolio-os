# portfolio-os

김대현(KimDaehyun, [@Swxp714](https://github.com/Swxp714)) — 기획·개발 올라운더 포트폴리오.

Windows 98 감성의 데스크탑 OS 인터페이스로 구성한 인터랙티브 포트폴리오 사이트 + 1페이지 이력서.

- **사이트**: `index.html` — 드래그 창 · 작업표시줄 · CRT 모니터 효과 · Win98 아이콘
- **이력서**: `resume.html` — A4 1페이지, 인쇄 시 PDF 최적화

## 로컬 실행
```
# 정적 서버가 필요합니다 (ES 모듈 X, 단순 정적)
python -m http.server 8777
# http://localhost:8777
```
또는 `start.bat` 더블클릭.

## 스택
순수 HTML / CSS / JavaScript · Pretendard · Galmuri(픽셀) · Font Awesome · win98icons · Devicon
